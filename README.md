# Data Cleaning & Reporting Automation

## Project Overview

This project automates data cleaning and reporting workflows using Python, Excel, and Power BI.

## Key Features

- Handles missing values
- Removes duplicate records
- Standardizes inconsistent data
- Generates cleaned Excel files automatically
- Creates automated summary reports
- Visualizes data using Power BI

## Tools & Technologies

- Python
- Pandas
- Excel
- Power BI

## Project Structure

- `data/` - Raw dataset
- `output/` - Cleaned data and automated reports
- `Script/` - Python automation scripts
- `PowerBI/` - Power BI dashboard

## Dashboard Features

- Total Sales KPI
- Total Customers KPI
- Sales by City Chart
- Sales Summary Table

## How to Run

1. Install the required libraries:
   `pip install pandas openpyxl`

2. Run the data cleaning script:
   `py -3.13 data_cleaning.py`

3. Run the report generation script:
   `py -3.13 report_generation.py`

4. Open the Power BI dashboard to view the visual report.
